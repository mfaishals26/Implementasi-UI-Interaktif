## ⚙️ Installation

1. Clone this repository

   ```bash
   git clone https://github.com/yourusername/Implementasi-UI-Interaktif.git
   ```
2. Navigate into the folder

   ```bash
   cd Implementasi-UI-Interaktif
   ```
3. Install dependencies

   ```bash
   npm install
   ```
4. Start the project

   ```bash
   npx react-native start
   ```
